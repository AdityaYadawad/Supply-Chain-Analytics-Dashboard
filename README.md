Supply Chain Analytics Dashboard | Power BI · SQL · Python · Excel

An end-to-end supply chain and inventory analytics project focused on monitoring inventory levels, supplier performance, logistics efficiency, and operational KPIs. The project uses Power BI for interactive dashboards, SQL for querying and aggregating transactional data, Python for data preprocessing and analysis, and Advanced Excel for reporting and validation. Designed to simulate real-world supply chain decision-making with actionable insights for inventory optimization and delivery performance improvement.
